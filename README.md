# Simon Says Game 
A browser-based Simon Says game built with HTML, CSS, and JavaScript.

## Description
This project is a simple Simon Says memory game.
Players have to repeat sequences of colors or buttons, which get longer each round.
The game focuses on UI design, interactivity, and JavaScript logic.

## Features
- Clickable colored buttons
- Sequence memory challenge
- Tracks level and score
- Interactive and fun browser game

## Tech Stack
- HTML5
- CSS3
- JavaScript
